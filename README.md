# SmartVision AI

SmartVision AI is a Computer Vision and Deep Learning project focused on building intelligent visual recognition systems using AI models and modern machine learning frameworks.

The project leverages:

- TensorFlow for deep learning
- Ultralytics YOLO for object detection
- OpenCV for image processing
- MLflow for experiment tracking and model management

---

# 🚀 Features

- Real-time object detection
- AI-powered computer vision pipeline
- Deep learning model training and inference
- Experiment tracking with MLflow
- Dataset management
- Modular source code structure

---

📦 Installation
1️⃣ Clone the Repository
git clone https://github.com/harishgundapuUD/smartvision_AI.git
cd smartvision_AI
2️⃣ Create Virtual Environment (Recommended)
Windows
python -m venv venv
venv\Scripts\activate
Linux / Mac
python3 -m venv venv
source venv/bin/activate
3️⃣ Install Dependencies
pip install -r requirements.txt
▶️ Running the Project

Run the main application:

python -m streamlit run .\app.py
